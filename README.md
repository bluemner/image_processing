# image_processing
Image processing 

## hw1

`bin\hw1<ext> data\input.pgm bin\output.pgm scale<double>`

Files will be saved in bin folder.

Note this application has been build for x64 bit architecture 

Linux 
Run: ./run.sh

Windows 
you need to have vs2013+ installed
build: build.bat
run: run.bat
